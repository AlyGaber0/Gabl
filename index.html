Gabl

An interpreted programming language written in C from scratch.

fn fib(n) {
  if(n == 0) {
    return 0
  }
  if(n == 1) {
    return 1
  }
  return fib(n - 1) + fib(n - 2)
}

print(fib(10))

Lexer

Scans raw source text character by character and produces a stream of labeled tokens — identifiers, keywords, literals, and operators — that the parser can reason about structurally.

Parser

Uses recursive descent parsing to build an abstract syntax tree that encodes operator precedence and program structure, turning a flat token stream into a hierarchical representation of the program.

Evaluator

Walks the AST recursively and executes each node, using a scoped environment with parent chain lookup for variable resolution — enabling closures, function calls, and block scoping without a separate compilation step.
